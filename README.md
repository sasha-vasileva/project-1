# Узнай, високосный ли твой год

Пример эталонного Flask‑приложения, демонстрирующего best practices: приложение создаётся через фабрику `create_app()`, бизнес‑логика вынесена в сервис, и используется Flask‑Babel для i18n (англ/рус).

Структура проекта:

- `run.py` — точка входа для запуска приложения
- `requirements.txt` — зависимости
- `app/` — пакет приложения
  - `__init__.py` — фабрика приложения и инициализация расширений
  - `config.py` — конфигурация
  - `routes.py` — контроллеры (Flask routes)
  - `services/calculator_service.py` — бизнес‑логика (вынесена и НЕ зависит от Flask)
  - `templates/index.html` — HTML шаблон
  - `static/css/style.css` — стили
- `translations/` — каталоги переводов (po-файлы)
- `tests/` — простые тесты (unit + integration)

Как запустить:

1. Установите зависимости (рекомендуется виртуальное окружение):

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. (Опционально) Скомпилируйте сообщения переводов, чтобы увидеть интерфейс на русском/английском:

```bash
pybabel compile -d translations
```

3. Запустите приложение:

```bash
python run.py
```

4. Откройте в браузере:

http://127.0.0.1:5000/

Примеры использования:

```bash
# из командной строки
python run.py
# затем в браузере открыть 127.0.0.1:5000
```

Тесты:

```bash
pytest -q
```

Примечания:
- Ноль и отрицательные числа считаются некорректными: приложение вернёт локализованное сообщение `Некорректный год. Введите год от 1 и выше`.
